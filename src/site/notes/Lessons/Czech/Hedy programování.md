Doporučený věk: 10-15
Potřebné dovednosti: Čtení, psaní, používání prohlížeče na jakémkoli zařízení
Pomůcky: Zařízení s prohlížečem
___
Toto je úvod do programování pomocí programovacího jazyka Hedy.
Hedy je jazyk vytvořený pro výuku. Jak postupujete, učíte se používat správnou syntaxi a pokročilé funkce, dokud se jazyk pomalu nepromění v Python.
___
Python je velmi známý jazyk, a přestože není tak výkonný a efektivní jako některé jiné, je díky své jednoduchosti skvělým jazykem pro začátečníky.
Používá se k vytváření pluginů/doplňků/botů pro mnoho platforem, jako jsou:
- Discord (boti)
- Blender 3D (skriptování)
- Krita/GIMP (pluginy)
- Minecraft Java (Mody se dělají v Javě, ale existuje mnoho modů, které vám umožní použít Python ke skriptování věcí do hry)
- Sims 4 (mody)
A mnoho velkých společností používá Python dodnes pro práci na backendu (např. YouTube, Spotify...).
___
Po tomto krátkém úvodu jděte na [https://hedy.org/hedy](https://hedy.org/hedy) a postupujte podle instrukcí lekcí.
